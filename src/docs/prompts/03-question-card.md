# QuestionCard

## Objetivo

Criar o componente responsável por representar cada pergunta da pesquisa.

## Contexto

Cada pesquisa pode possuir múltiplas perguntas configuráveis.

## Prompt Utilizado

Preciso criar um componente React chamado QuestionCard.

Esse componente deve permitir:

- editar enunciado
- remover pergunta
- alterar tipo da pergunta
- habilitar pergunta condicional

Estou utilizando React e CSS modularizado.

## Resultado Obtido

Foi criada uma estrutura reutilizável para gerenciamento das perguntas.

## Ajustes Realizados

Posteriormente foram adicionados novos tipos de perguntas e integração com o modal de seleção de tipo.

## Impacto no Projeto

O QuestionCard tornou-se o componente central do editor de pesquisas.
