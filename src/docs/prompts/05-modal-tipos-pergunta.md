# Modal de Tipos de Pergunta

## Objetivo

Permitir seleção visual do tipo de pergunta.

## Contexto

O usuário precisava configurar diferentes formatos de resposta.

## Prompt Utilizado

Preciso criar um modal React mobile-first para seleção de tipos de pergunta.

Os tipos disponíveis são:

- Texto
- NPS
- CSAT
- Badge/Chips

O modal deve ser reutilizável e simples de utilizar.

## Resultado Obtido

Foi criada uma interface dedicada para seleção dos tipos.

## Ajustes Realizados

Posteriormente foi reestilizada para ser responsiva em telas mobile.

## Impacto no Projeto

A experiência de configuração das perguntas tornou-se mais intuitiva.
