# Autenticação Local

## Objetivo

Implementar login e cadastro sem backend.

## Contexto

O projeto precisava de autenticação funcional para demonstração do MVP.

## Prompt Utilizado

Preciso criar um sistema de cadastro e login utilizando Local Storage.

Requisitos:

- armazenar usuários localmente
- criptografar senha
- validar login
- manter sessão ativa

## Resultado Obtido

Foi criado um sistema simples de autenticação baseado em armazenamento local.

## Ajustes Realizados

Foram realizados ajustes para persistência de sessão e redirecionamentos.

## Impacto no Projeto

Permitiu demonstrar o fluxo completo da aplicação sem necessidade de backend.
