# Perguntas Condicionais

## Objetivo

Permitir que perguntas sejam exibidas com base em respostas anteriores.

## Contexto

O sistema precisava suportar lógica condicional entre perguntas.

## Prompt Utilizado

Quero implementar perguntas condicionais.

Quando o usuário ativar essa opção, devem ser exibidas configurações adicionais para definir:

- pergunta de origem
- condição
- valor esperado

A interface deve permanecer simples e intuitiva.

## Resultado Obtido

Foi criada uma estrutura específica para configuração de condições.

## Ajustes Realizados

Foram adicionadas validações para impedir condições vazias.

## Impacto no Projeto

A funcionalidade aumentou significativamente a flexibilidade das pesquisas.
