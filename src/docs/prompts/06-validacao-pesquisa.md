# Validação de Pesquisas

## Objetivo

Garantir consistência dos dados antes da publicação.

## Contexto

Era necessário impedir a publicação de pesquisas incompletas.

## Prompt Utilizado

Preciso validar uma pesquisa antes de permitir sua publicação.

Regras:

- título obrigatório
- pelo menos uma pergunta
- perguntas condicionais devem possuir configuração válida

## Resultado Obtido

Foi criado um sistema de validação centralizado.

## Ajustes Realizados

Novas regras foram adicionadas conforme surgiram novos tipos de perguntas.

## Impacto no Projeto

Redução de inconsistências e melhoria da qualidade dos dados.
